Este sitio ha sido creado con el proposito de elaborar recursos educativos digitales. Con el tema de diseño de sitios web como son las etiquetas básicas de HTML, los atributos y valores, temas importantes para empezar en el mundo del diseño web.

Etiquetas básicas de HTML

HTML es un markup language, lo que significa que está escrito con códigos que puede leer una persona sin que sea necesario compilarlo primero. En otras palabras, el texto en una página web está «marcado» con estos códigos para dar instrucciones al navegador web sobre cómo mostrar el texto. Estas etiquetas de marcado son las propias etiquetas HTML.
Todas las etiquetas HTML están hechas con un número de partes específicas, incluyendo: El carácter “menor que” (<) Cualquier número de atributos HTML que se quiera usar, El carácter “mayor que” (>)

Cuáles son las etiquetas HTML básicas

  • body para el contenido.
  • headpara información sobre el documento.
  • apara enlaces.
  • b para poner el texto en negrita.
  • br para saltos de línea.
  • h1..h2.. para títulos dentro del contenido.
  • ul para listas desordenadas, li para elementos dentro de la lista.
  • p para parágrafos.

    • Atributos y Valores

    Los atributos son elementos de las etiquetas que pueden proporcionar ciertas características a un texto, imagen, etc. (por ejemplo: altura, ancho, color, etc.)
    Van insertados dentro de la etiqueta de inicio o apertura y llevan inmediatamente detrás el signo (“=”) con los valores de dichos atributos.
    HTML no distingue entre mayúsculas y minúsculas, por eso, cuando es importante hacerlo, cosa que puede ocurrir con los valores de algunos atributos, es preciso encerrar el valor entre comillas dobles. El punto y como que sigue al atributo se usa para separar los diferentes comandos de estilo.
