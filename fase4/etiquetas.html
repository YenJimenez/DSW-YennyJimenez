Etiquetas Basicas de HTML

HTML es un markup language, lo que significa que está escrito con códigos que puede leer una persona sin que sea necesario compilarlo primero. En otras palabras, el texto en una página web está «marcado» con estos códigos para dar instrucciones al navegador web sobre cómo mostrar el texto. Estas etiquetas de marcado son las propias etiquetas HTML.
Todas las etiquetas HTML están hechas con un número de partes específicas, incluyendo: El carácter “menor que” (<) Cualquier número de atributos HTML que se quiera usar, El carácter “mayor que” (>). XAVIER IDEVIK. (2021)

Cuáles son las etiquetas HTML básicas

  • body para el contenido.
  • head para información sobre el documento.
  • a para enlaces.
  • b para poner el texto en negrita.
  • br para saltos de línea.
  • h1..h2.. para títulos dentro del contenido.
  • ul para listas desordenadas, li para elementos dentro de la lista.
  • p para parágrafos.

    • HTML consiste en una serie de elementos que usarás para encerrar diferentes partes del contenido para que se vean o comporten de una determinada manera. Las etiquetas de encierre pueden hacer de una palabra o una imagen un hipervínculo a otro sitio, se pueden cambiar palabras a cursiva, agrandar o achicar la letra, etc.MDN contributors.(2020)

    imagen anatomia de un elemento HTML. MDN contributors.(2020)

    Las partes principales del elemento son:

  • La etiqueta de apertura: consiste en el nombre del elemento (en este caso, p), encerrado por paréntesis angulares (<>) de apertura y cierre. Establece dónde comienza o empieza a tener efecto el elemento —en este caso, dónde es el comienzo del párrafo—.
  • La etiqueta de cierre: es igual que la etiqueta de apertura, excepto que incluye una barra de cierre (/) antes del nombre de la etiqueta. Establece dónde termina el elemento —en este caso dónde termina el párrafo—.
  • El contenido: este es el contenido del elemento, que en este caso es sólo texto.
  • elemento: la etiqueta de apertura, más la etiqueta de cierre, más el contenido equivale al elemento.
